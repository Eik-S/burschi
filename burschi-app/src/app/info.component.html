The Graph

  1. We extracted all the links to Burschenschaften from burschenschaft.de.
  2. We looked up all the links on each website which route to another domain.
  3. We added directed links if the source page has a link to the target page.

We exclude all external nodes which have just one connection from the visualisation.

Nodes are scaled by their degree in the network.
